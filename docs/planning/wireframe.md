**경유 표시**

```
│ 10:00     ─── 경유 1회 ───      18:30    │
│  ICN        PVG (2시간)          NRT     │
│          8시간 30분                       │
```

| 요소 | 설명 |
|------|------|
| 항공사 로고 | `owner.logo_symbol_url` (Duffel) 또는 IATA 코드 기반 로고 URL |
| 출발/도착 시간 | `HH:mm` 형식, 현지 시간 |
| 공항 코드 + 이름 | IATA 3자리 + 도시명 |
| 소요 시간 | `N시간 M분` |
| 경유 정보 | `직항` 또는 `경유 N회` + 경유지 코드 + 대기 시간 |
| 가격 | KRW, 천 단위 구분자 (₩185,000) |
| 데이터 출처 | `Duffel` 또는 `Kiwi` 뱃지 |
| 선택 버튼 | Kiwi → deep_link로 외부 이동, Duffel → 상세 페이지 (MVP에서는 정보만) |

|------|--------|

| API 부분 장애 | "일부 항공사 결과가 누락될 수 있습니다" (상단 배너) |
| API 전체 장애 | "검색에 실패했습니다. 잠시 후 다시 시도해주세요" + 재시도 버튼 |
| 네트워크 오프라인 | "인터넷 연결을 확인해주세요" |

### 빈 검색 폼 유효성

| 필드 | 유효성 검사 |
|------|------------|
| 출발지 | 필수, IATA 코드 유효성 |
| 도착지 | 필수, 출발지와 다를 것 |
| 날짜 | 과거 날짜 불가, 출발일 ≤ 귀국일 |
| 승객 | 1~9명 |

## 5. 컴포넌트 구조 (React)

```
src/
├── components/
│   ├── search/
│   │   ├── SearchForm.tsx           # 검색 폼 전체
│   │   ├── AirportInput.tsx         # 공항 자동완성 입력
│   │   ├── SearchModeTab.tsx        # 날짜 지정 / 유연한 날짜 탭
│   │   ├── FlexibleDateInputs.tsx   # 요일/숙박/범위 입력
│   │   ├── FixedDateInputs.tsx      # 출발일/귀국일 달력
│   │   └── PassengerInput.tsx       # 승객 수 + 좌석 등급
│   ├── results/
│   │   ├── FlightCard.tsx           # 항공편 카드
│   │   ├── FlightCardSkeleton.tsx   # 로딩 스켈레톤
│   │   ├── FlightList.tsx           # 카드 리스트 + 정렬
│   │   ├── FilterSidebar.tsx        # 필터 사이드바 (Desktop)
│   │   ├── FilterBottomSheet.tsx    # 필터 바텀시트 (Mobile)
│   │   └── SearchSummary.tsx        # 검색 조건 요약 바
│   ├── flexible/
│   │   ├── DateCombinationCard.tsx   # 날짜 조합 카드
│   │   └── DateCombinationList.tsx   # 조합 리스트 + 정렬
│   └── common/
│       ├── Header.tsx               # 상단 헤더
│       ├── ErrorState.tsx           # 에러 표시
│       └── PriceBadge.tsx           # 가격 표시 (₩ 포맷)
├── api/
│   ├── flights.ts                   # 항공편 검색 API 호출
│   ├── flexibleSearch.ts            # 유연한 날짜 검색 API 호출
│   └── airports.ts                  # 공항 자동완성 API 호출
├── types/
│   ├── flight.ts                    # FlightOffer, FlightSearchRequest
│   ├── flexible.ts                  # FlexibleSearchRequest/Result
│   └── airport.ts                   # Airport 자동완성
└── hooks/
    ├── useFlightSearch.ts           # 검색 상태 관리
    ├── useAirportSearch.ts          # 자동완성 디바운스
    └── useFlightFilter.ts           # 필터/정렬 로직
```
