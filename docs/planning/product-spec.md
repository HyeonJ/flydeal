# 플라이딜 제품 기획서

## 1. 제품 개요

**플라이딜(FlyDeal)** — 한국어 AI 자연어 검색 기반 항공편 가격 비교 서비스

"4월에 도쿄 2박3일 제일 싼 거" 같은 자연어로 항공편을 검색하고, 최저가 알림을 받을 수 있는 서비스.

## 2. 문제 정의

- 기존 항공편 검색(스카이스캐너, 구글 플라이트)은 출발지/도착지/날짜를 정확히 입력해야 함
- 한국어 자연어 항공편 검색 서비스가 없음 (Skyscanner ChatGPT는 영어 UK/US 한정)
- 한국 LCC(제주항공, 진에어, 티웨이, 에어부산, 에어서울) 통합 가격 추적 서비스 부재
- 가격 변동 알림이 세밀하지 않음 (특정 항공사+직항+날짜 범위 조합 설정 불편)

## 3. 타겟 사용자

- 가격에 민감한 20~30대 여행자
- 단거리 노선(일본, 대만, 동남아) 자주 이용
- "언제가 제일 싼지" 알고 싶은 유연한 일정 여행자

## 4. 핵심 기능

### MVP (1단계)
1. **유연한 날짜 검색 (핵심 기능)** — 요일 패턴 + 기간으로 최저가 날짜 조합 탐색
   - 예: "2026년, 금~월, 3박4일, 도쿄" → 모든 금~월 조합 중 최저가순 정렬
   - 예: 5/22~25 ₩180,000 | 6/5~8 ₩195,000 | 7/3~6 ₩210,000 ...
   - Kiwi API `date_from/date_to` 범위 검색 활용
2. **항공편 검색** — Duffel API + Kiwi Tequila API 이중 연동
   - 대한항공/아시아나: Duffel (GDS)
   - 한국 LCC: Kiwi Tequila (제주항공, 진에어, 티웨이, 에어부산, 에어서울)
3. **검색 결과 표시** — 가격순/시간순 정렬, 직항/경유 필터
4. **기본 UI** — 검색 폼 + 결과 리스트 + 항공편 카드

### 2단계
5. **AI 자연어 검색** — "4월 셋째주 도쿄 제일 싼 거" → 자동 파싱
   - Claude Tool Use로 한국어 자연어 → 구조화된 검색 파라미터 변환
6. **가격 알림** — 경로/날짜 등록 → 가격 하락 시 알림
   - Cron 기반 주기적 가격 체크
   - 이메일/푸시 알림

### 3단계 (장기)
7. **가격 히스토리** — 노선별 가격 추이 차트
8. **가격 예측** — "지금 사 vs 기다려" 표시
9. **핀테크** — Price Freeze (가격 동결 보증) 한국 적용

## 5. 차별화 포인트

| 기존 서비스 | 플라이딜 차별화 |
|------------|---------------|
| 날짜 고정 검색만 지원 | **요일 패턴+기간으로 최저가 날짜 자동 탐색** |
| 영어 자연어 검색만 (Skyscanner+ChatGPT) | **한국어 자연어 검색** |
| 한국 LCC 통합 검색 약함 | **Kiwi API로 한국 LCC 5개사 커버** |
| 기본 요금만 표시 | LCC 총비용 표시 (기본+수하물+좌석) |
| 가격 알림 조건 설정 불편 | 세밀한 조건 조합 (항공사+직항+날짜 범위) |

## 6. 수익 모델

1. **Duffel 커미션** — 항공사 커미션 공유 (판매량 비례, 초기비용 없음)
2. **Kiwi 어필리에이트** — 예약 시 3% 커미션
3. **Duffel Links** — 마크업 직접 설정 가능
4. **(장기) 핀테크 상품** — Price Freeze, 유연 취소 등

## 7. 경쟁 환경 요약

- **Google AI Mode**: 검색→예약 원스톱, 글로벌 최대 위협. 하지만 한국 LCC 약함
- **Skyscanner + ChatGPT**: 대화형 검색 프로덕션 출시. 영어 한정
- **네이버 항공**: 한국 검색 63% 점유. AI 검색 미도입
- **마이리얼트립**: MAU 500만+, AI 도입 시작. 항공 전문 아님
- **NUUA**: NDC/GDS 통합 API B2B. 소비자 시장 빈자리 존재

**전략**: 대형 플랫폼과 정면 경쟁 X → 한국 LCC 특화 + 한국어 AI로 틈새 공략

## 8. 기술 아키텍처

```
[사용자] → [React Frontend]
              ↓
         [Spring Boot API]
              ↓
    ┌─────────┼─────────┐
    ↓         ↓         ↓
[Duffel]  [Kiwi]   [MySQL/Redis]
(FSC/NDC) (LCC)    (캐시/히스토리)
```

### API 이중 전략
- **Duffel**: 대한항공, 아시아나 (GDS/NDC 직접 연결, DX 최고)
- **Kiwi Tequila**: 한국 LCC + 가상 인터라이닝 (최저가 다구간)

### 주요 기술 결정
- Duffel 커미션 모델 → 초기비용 없이 시작 가능
- Kiwi API로 한국 LCC 커버리지 확보 (Duffel 미지원)
- Redis로 검색 결과 캐싱 (Offer 만료: 30분~24시간)
- Amadeus Self-Service API 2026.07 종료 → Duffel/Kiwi 선택 정당화
